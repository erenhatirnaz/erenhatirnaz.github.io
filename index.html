who am i?

I'm a computer engineer. I do mostly software development. Nowadays I'm calling myself as a "Back-End Developer" but I'm also interested in Dev-Ops Engineering as a future career plan. Simplicity is always on my mind when I'm coding. I'm also believe in free and open source software philosophy. My favorite part of my profession is that it will never became monotonous. Because this profession requires following and teaching new technologies or approaches, and I'm in love with learning new things!

Emacs is my daily driver. Major technologies I use are: PHP, JavaScript, Bash, Ruby, Git and GNU/Linux. Some of my completed software projects are listed in the projects section below. Visit my GitHub or GitLab profile for my all projects.

One of my greatest passions is: Writing. On my blog, you can find my articles on programming, book reviews and many other topics, mostly written in Turkish. About 2-3 years ago, I was a more active writer. Although I don't publish most of them, I also continue to write these days. But I really want to go back to blog posts as soon as possible. Hopefully I can do it.

projects

# semantic-video-search

web, academic | published, doi: 10.1007/s11042-020-08659-2 | source code not available

This is my graduation thesis and my academic work with my teacher and advisor from my second university. In this project, we brought together Image Processing and Semantic Web. I have developed a web interface that provides detailed video search for users. I also conducted user experiments to show the difference between keyword-based search methods and semantic video search. Our article was published in the 79th issue of Multimedia Tools and Applications Journal published by Springer. Major technologies I used in this project are: JavaScript, VueJS, Apache Jena Fuseki, RDF & SPARQL. Unfortunately, the source coude of the project isn't public.

# mgm-radar

cli | source code: github

A command-line application that downloads and shows meteorological radar images from mgm.gov.tr (Turkey's national meteorology agency). I was wrote this tool for my own needs and now I'm using it almost everyday. I'm interested in meteorology. This tool written in Bash and works well on GNU/Linux distros.

# w3c-sql-editor-with-turkish-database

web | source code: github

A PHP script that some kind of clone of W3C's Tryit Editor but with support Turkish database. I developed this project when I was at my first university. In database design lesson, our teacher was using W3C's editor. The editor's database was designed and populated entirely in English, but there was a language problem in the classroom. So I developed this project to help my teacher and friends. As far as I know, my teacher still uses this project in classrooms. I'm very happy to help a lot of people who want to learn database design. I used SQLite as the relational database in this project.

# anger.co-startup-parser

cli, academic | source coude not available

I developed this project for my teacher from my first university. A Ruby script that takes the marketplaces and locations of all startup companies working in the IoT industry from angel.co and generates a CSV file with that information. This CSV file was then converted into a graph. My teacher used this graph in his presentation at the International Conference on Information Technology and Science in Japan in 2016. Unfortunately, the source coude of the project isn't public.
